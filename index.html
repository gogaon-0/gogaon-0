자동 조정

서버 규칙 위반을 자동으로 감지하고 조치합니다. 욕설 필터링, 스팸 방지, 자동 경고 시스템으로 건강한 커뮤니티를 유지하세요.

🚫

블랙리스트 시스템

문제 사용자를 자동으로 차단하고 관리자에게 알림을 전송합니다. 블랙리스트 사용자가 서버 참가 시 즉시 킥 처리됩니다.

💬

금지어 필터

대시보드에서 금지어를 설정하면 자동으로 메시지를 삭제하고 경고합니다. 깨끗한 채팅 환경을 유지하세요.

👋

환영 메시지

새로운 멤버를 자동으로 환영하고 서버 규칙을 안내합니다. 맞춤형 환영 메시지로 친근한 첫인상을 만들어보세요.

👤

퇴장 알림

멤버가 서버를 떠날 때 자동으로 알림을 전송합니다. 누가 언제 떠났는지 쉽게 추적할 수 있습니다.

🎟️

티켓 시스템

버튼 클릭으로 간편하게 1:1 문의 채널을 생성합니다. 체계적인 고객 지원이 가능합니다.

📢

공지 시스템

대시보드에서 직접 공지를 작성하고 전송할 수 있습니다. @everyone 멘션 옵션도 제공됩니다.

⚙️

웹 대시보드

Discord로 로그인하여 브라우저에서 모든 설정을 관리하세요. 명령어 없이도 쉽게 봇을 제어할 수 있습니다.

ModuCore 봇에 대하여

ModuCore 디스코드 봇은 서버 관리자들이 더 효율적으로 커뮤니티를 운영할 수 있도록 설계되었습니다. 블랙리스트 관리, 금지어 필터링, 자동 환영/퇴장 메시지, 티켓 시스템, 공지 전송 등 모든 기능을 웹 대시보드에서 간편하게 설정할 수 있습니다. Discord OAuth2로 로그인하면 관리자 권한이 있는 서버만 표시되며, 안전하고 직관적인 인터페이스로 누구나 쉽게 사용할 수 있습니다. 지금 바로 봇을 추가하고 더 나은 서버 관리 경험을 시작해보세요!
